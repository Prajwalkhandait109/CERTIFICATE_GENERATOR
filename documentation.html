Installation

Step 1: Download

Download the latest version of Certificate Generator from the download page.

Step 2: Extract

Extract the ZIP file to a folder on your computer. Right-click the ZIP file and select "Extract All".

Step 3: Run

Double-click START_HERE.bat or CertificateGenerator.exe to launch the application.

Windows Security Warning

Windows may show a security warning. Click "More info" then "Run anyway". This is normal for unsigned applications.

Quick Start

  1. Launch the Application

    Double-click the executable. Your browser will open automatically to http://localhost:5000

  2. Upload Files

    Upload your names file (CSV/Excel) and certificate template (PNG/JPG)

Complete User Guide

Preparing Your Files

Names File

Create a CSV or Excel file with a column named "name":

name
John Smith
Sarah Johnson
Michael Brown
Emily Davis

Certificate Template

  • Use PNG or JPG format
  • Recommended size: 1200x800 pixels or larger
  • Leave blank space where names will appear
  • Ensure good contrast for text visibility

Using the Application

1. Upload Files

Click "Choose File" for both the names file and template image.

2. Customize Settings

  • Font Size: 10-200 pixels (default: 60)
  • Text Color: Choose from presets or enter custom color
  • Position: X and Y coordinates for text placement
  • Font Family: Arial, Times New Roman, etc.
  • Output Format: PNG or PDF

3. Generate Certificates

Click "Generate Certificates" and wait for processing to complete.

4. Download

Download certificates individually or as a ZIP file containing all certificates.

Customization Options

Font Settings

  • Size: Adjust based on template size (40-80px for most templates)
  • Family: Choose from system fonts
  • Color: Use contrasting colors for readability

Position Settings

The position is specified in X (horizontal) and Y (vertical) coordinates:

  • X: Distance from left edge (0 = left, higher = right)
  • Y: Distance from top edge (0 = top, higher = bottom)
Pro Tip

Start with the center of your template and adjust in small increments (10-20 pixels) until perfect.

Tips & Tricks

Test First

Generate one certificate first to verify positioning before processing all names.

High Resolution

Use high-resolution templates (300 DPI) for professional printing quality.

Contrast Matters

Ensure text color contrasts well with the background for readability.

Longest Name

Test with the longest name in your list to ensure it fits properly.

Troubleshooting

Application Won't Start

  • Check Windows security warning and click "Run anyway"
  • Ensure port 5000 is not in use by another application
  • Try running as administrator
  • Check antivirus isn't blocking the executable

Browser Doesn't Open

  • Wait 5-10 seconds for the server to start
  • Manually open http://localhost:5000 in your browser
  • Check firewall settings

Files Won't Upload

  • Verify file format (CSV/XLSX for names, PNG/JPG for template)
  • Check file size is under 16 MB
  • Ensure files are not corrupted

Names Not Appearing

  • Ensure CSV/Excel has "name" column header
  • Check for empty rows in your file
  • Verify names are in the first column

Text Position Wrong

  • Adjust X coordinate to move left/right
  • Adjust Y coordinate to move up/down
  • Make small adjustments (10-20 pixels at a time)

Frequently Asked Questions

Do I need Python installed?

No! The executable includes everything you need. Just download and run.

Is my data safe?

Yes! All processing happens locally on your computer. No data is sent to any server or cloud service.

Can I use custom fonts?

Currently, the application uses system fonts. Custom font support may be added in future versions.

How many certificates can I generate at once?

Unlimited! You can generate as many certificates as you need in a single batch.

Does it work offline?

Yes! Once downloaded, the application works completely offline without an internet connection.

Can I use it for commercial purposes?

Yes! The application is free to use for both personal and commercial purposes.
